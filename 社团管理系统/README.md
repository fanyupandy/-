# 社团管理系统

一个基于 `Flask + SQLite + SQLAlchemy` 的简易社团管理网站，支持社团资料维护、成员管理和职位分配，适合作为课程作业、小型练习项目或 Flask 入门项目。

## 项目简介

本系统用于完成社团的基础信息管理，主要围绕“社团”和“成员”两类数据展开。用户可以在页面中创建社团、编辑社团简介与头像、添加成员，并为成员设置不同职位。

项目当前采用单文件结构，前后端代码集中在 `app.py` 中，启动简单，适合本地快速运行与演示。

## 功能特点

- 创建社团
- 编辑社团名称、简介和头像
- 删除社团
- 添加成员到指定社团
- 修改成员职位
- 将成员移出社团
- 使用 SQLite 持久化存储数据
- 页面可直接在浏览器中进行可视化操作

## 技术栈

- Python
- Flask
- Flask-SQLAlchemy
- SQLite
- HTML / CSS / JavaScript

## 运行环境

建议环境：

- Python 3.10 及以上
- Windows / macOS / Linux

## 安装依赖

如果本地还没有安装相关库，可以先执行：

```bash
pip install flask flask-sqlalchemy
```

## 启动项目

在项目根目录下运行：

```bash
python app.py
```

启动成功后，默认访问地址为：

```text
http://127.0.0.1:5000
```

## 使用说明

1. 启动项目后，在浏览器打开首页。
2. 点击“新建社团”创建社团。
3. 选择左侧社团后，可以修改社团名称、简介和头像。
4. 在右侧表单中添加成员，并设置其职位。
5. 可以在成员表格中修改职位，或将成员移出社团。
6. 如需删除整个社团，可点击“删除当前社团”。

- 如果数据库文件损坏或异常，可删除后重新运行项目自动生成新库。
